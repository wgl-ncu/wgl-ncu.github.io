#标准库类型string#
定义与初始化：```string s1; string s2=s1; string s3="hiya"; string s4("hiya"); string s5(10,'c');``` 使用等号是拷贝初始化，不使用等号是直接初始化 string操作，p77 读写string对象： cin>>s;//遇到空格停止，如：“ abc de”，s="abc". getline(cin,s);//遇到换行结束，会将换行符读入，但是不会将换行符写出。 如：“ abc de”，s=" abc de". while(cin>>s)cout<...... string对象相加：两个字符串相加，得到一个新的字符串。 string和字面值相加：必须保证每一个＋的两边至少有一个string ```string s=s1+s2+'aaa';√ string s='aaa'+'bbb'+s2;× string s='aaa'+s2+'bbb';```√从左到右依次相加
#处理string对象中的字符# cctype头文件，p82 处理所有字符：范围for语句for(auto c:s)将s中的每一个字符依次转化为c。for(auto c,s)cout< vector是模板，用<>来指定vector实例化成为什么样的类型。注意：类型不能是引用。 定义和初始化vector类型对象： vector v1/v1(v2)/v1=v2...... vectorv(unsigned x，T s);圆括号里为元素个数，s为初始化的元素，省略则默认初始化。 vectorv{Ts...};花括号为元素，写几个元素就包含几个元素。 对于string类型，vectorv{10,"cd"}==vectorv(10,"cd");//表示有10个cd。 列表初始化：vectorv1{......};//只能使用花括号初始化。 向vector中添加元素：push_back成员函数，将目标元素压入vector类型的尾部。v.push_back(1);//将1压入v的尾部。 vector对象不建议定义大小。 vector操作：类似string，不能使用下标来添加元素。 为确保下标合法化建议尽可能使用范围for语句。 #迭代器# auto b=v.begin(),c=v.end();//begin指向第一个元素，end指向最后一个元素的下一位置，成为尾迭代器。 如果容器为空，begin和end返回的是同一个迭代器，都是尾迭代器。 对容器使用for语句时需要使用！=而非<,>因为容器元素并非支持<,>运算。 迭代器的运算符：*iter；返回迭代器所指元素的引用。 ++iter，--tier。 c++11引入cbegin和cend专门获得常量类型的迭代器。 结合解引用：(*it).empty()==it->empty()，it所指的对象是否为空。 向一个容器中添加了元素，会使其迭代器失效。 迭代器运算：+，-，>,<...... #数组# 数组大小确定不变。 定义与初始化：数组的维度必须是常量表达式。int a[b];b必须是const unsigned而不能是unsigned 不允许使用引用的数组。int&a[5];× 显示初始化：int a[3]={1,2,3}; int a[]={1,2,3}; 字符数组：可以用上述初始化方式和int c[]="hello";//c的长度为字符串长度加1，最后一格会加上一个空格。 不允许使用一个数组初始化或者赋值给另一个数组。 允许定义数组的指针以及数组的引用：int(*array)[10]=&arr;int(&array)[10]=arr;//从括号内向外，再从左向右。 在函数内部定义的数组默认初始化会给与不确定的值。 数组与指针：int *p=nums;==int *p &nums[0]; 一些情况下数组的操作是指针的操作。int ia[]={......};auto a2(ia); a2是指向ia首地址的指针。 但是decltype(ia) a3={......};是正确的，decltype返回的是数组类型。 指针也是迭代器：指针适用vector和string迭代器所支持的所有运算。 a[10]数组。可以用int *p=a[10];来获得指向a数组尾元素下以位置的指针。类似于end迭代器。 c++11引入begin和end函数来获取数组的头尾指针。头文件iterator int* first=begin(a);int* last=end(a); 下标和指针：只要指针指向数组中的元素，指针可以进行下标运算。int*p=&a[2];cout<<*(p+2); 利用数组初始化容器：int a[]={1,2,3,4,5};vectorv(begin(a),end(a)); #多维数组# c++没有多维数组，只有数组的数组。 int a[3][4]; 多维数组与指针：数组a中包含了3个元素，每个元素是拥有4个元素的数组。 所以 数组a的指针是：int (*p)[4]=a/begin(a);//p是指向包含4个整型数组的指针。 多维数组的初始化：int[3][[4]={ {1,2,3，3},{4,5,6，6},{7,8,9，9} };内层花括号非必须。 使用范围for语句时，如果在最内层之外不使用引用，auto得到的类型会是指针，程序会无法编译。
