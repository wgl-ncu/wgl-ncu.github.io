函数基础
定义：返回类型+函数名+形参+函数体
实参和形参：实参是形参的初始值。
形参列表：0~多个，可以用（）或者（void）来表示没有形参。
                （int a,int b）√；（int a,b）×；
函数的返回类型：数组和函数类型是不被允许的。可以用指向数组或者函数的指针代替。
局部对象：形参和函数体内定义的变量成为局部变量。
自动对象：只存在于块执行期间的对象。
局部静态对象：函数执行时创建，程序结束时才销毁。
函数声明：用；代替函数体，建议在头文件中声明函数。
分离式编译：允许我们将程序分割到几个文件中去，每个文件独立编译。、
参数传递
传值参数：非引用形参的改变不会影响原来实参的值。
指针形参：和非引用类型类似。函数执行后，指针只能改变所指对象的值，不能改变其地址。void result（int *p）{p=0;}//不能改变实参的地址。
    建议使用引用类型的形参代替指针。
传引用参数：函数中的引用形参绑定实参，对其的改变会同步改变实参。
                引用避免拷贝，可以提高效率。如果不需要改变实参，可以将形参定义为常量的引用。
const形参和实参：当用实参初始化形参时会忽略掉顶层const；
void(const int a)和void(int a)不能同时定义。
初始化回顾：const int &r2=42;√  int &r2=42;×  //不能用字面值初始化一个非常量引用，可以初始化常量引用。
指针或者引用的形参：非常量指针只能由对应的类型的指针实参来初始化。int find（int *p）->find(&a);==int *p=&a;
                                  非常量引用只能由对应的引用来初始化。 int find(int &p)->find(a);==int &p=a;
                                  常量引用可以用字面量来初始化。int find(const int &p)->find(20);==const int &p=20;
尽量使用常量引用：常量引用拥有比非常量引用更多的初始化方法。
数组形参：注意：数组不允许拷贝，数组在使用时通常会转化为指针。
                void sss(const int*)==void sss(const int[])==void sss(const int[10])//传入的参数是int*类型即可。
        
管理指针形参的办法：1.使用标记。例如字符数组最后一个字符后跟着的空字符
                                  2.使用标准库规范。传入数组的头和尾指针。
                                  3.显示传递一个表示数组大小的形参。
数组形参和const：不改变数组的元素的时候建议定义数组形参为指向const的指针。
数组引用形参：f(int (&a)[10]);//传入的参数必须为含有10个整数的数组。
传递多维数组：void p(int (*ss)[10])==void p(int ss[][10]);//形参是指向含有10个整数数组的指针。
含有可变形参的函数：initializer_list形参：适用于实参数量未知但是类型相同的函数。
                                    和vector一样，initializer_list是一个模板类型，初始化要说明元素类型：initializer_list<int> s;
                                    注意：initializer_list中的元素是常量值，不能改变。
                                    传递参数时，要将元素放在一个花括号中。
                                  省略符形参：仅仅用于c和c++通用类型。
返回类型和return语句
无返回值函数：return；//退出函数，只能用在返回类型式void的函数中。
有返回值函数：return experrion；//返回的类型必须和函数返回类型相同或能隐式转换。
必须保证函数最终都能通过一个return语句返回。
不要返回局部对象的引用或指针，因为他们会被释放而失去效果。
返回的如果是一个引用，那么返回的就是一个左值。其余为右值。
vector<string> poss(){
    return {"hello","world"};
}
递归：

函数直接或者间接调用自身就是递归。注意：main函数不能调用自己。递归时传递参数不要使用–i，++i；

返回数组指针：

可以使用类型别名来简化定义。如：using art=int[10];art* func(int a);func返回一个指向含有10个整数的数组的指针。
尾置返回类型：auto func(int a)->int(*)[10];
使用decltype：在知道返回的指针指向哪个数组时。
