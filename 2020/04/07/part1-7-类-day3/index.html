构造函数再探：
构造函数的初始值：const和引用类型必须初始化。当成员属于一个没有默认构造函数的时候也必须初始化。
建议多使用构造函数初始值。
初始化顺序：先在类中声明的先初始化，后声明的后初始化。
    class A{
    int x;int y
    A():y(1),x(y);//错误，x先声明，所以先执行x(y),出现未定义错误。
    };
构造函数如果为参数指定了默认初始值，则它就是默认构造函数。
    class A{
    public:
        A()=default;
        A(int i=0):b(i){}//这个函数和上面的函数都定义为默认构造函数。不能同时使用。
    private:
        int b;
        int c=1;
    };
委托构造函数：将构造职责委托给其他构造函数。
    class A{
    public:
        A(int x,int y,int z):a(x),b(y),c(z){};
        A():A(0, 0, 0){};
        A(int x):A(x,0,0){};//第二个和第三个构造函数将构造任务委托给了第一个。
    private：
        int a,b,c;
    }；
默认构造函数的作用：
    注意，默认构造函数调用时不要加括号，会被判定成为声明了一个函数。
    如果定义了一个类，没有默认构造函数也没有将其用其他的构造函数初始化的化，这个类对象就是无效的。
隐式类类型转换：
    编译器可以帮助我们进行一次隐式的类型转换。
    可以在构造函数前添加explicit来阻止隐式类类型转换。//使用该修饰词的时候类不能进行拷贝初始化。
聚合类：
    全员public，无构造函数，无类内初始值，无基类和virtual函数。
    聚合类可以直接访问其成员，并且可以用{}括起来的初始值列表来进行初始化。注意：初始值顺序必须和声明顺序一致。
字面值常量类：
    定义：只有字面值类型数据的聚合类。或者该类只有字面值类型数据，且含有至少一条constexpr构造函数，类内初始值必须是常量表达式。
    constexpr构造函数：必须初始化所有成员。
类的静态成员：
    通过在对象之前使用关键子static使得类与对象相关联。该对象被所有该类实例所共享。
    静态成员的使用：既可以向非静态成员一样被累的实例所调用，也可以直接使用作用域运算符访问。在类内可以直接被调用。
    定义静态成员：在类外定义时，和非静态成员一样。注意不要添加static关键字。
                            必须在类的外部定义和初始化静态成员。
    静态成员可以声明乘它所属的类类型，非静态之能声明为指向它所属类型的指针或者引用。
    静态成员可以作为默认实参。
    不要再主函数中定义。
    如果要使用类内初始值，必须是常量。
