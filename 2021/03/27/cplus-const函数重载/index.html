c++const函数重载

重载是什么

重载就是一个函数使用不同的参数列表，程序运行时会根据传递的参数类型来选择使用哪一个函数。
重载是在编译阶段已经完成，对于汇编和链接来说透明的。

class Test {
public:
    int A(int& a){return a;}
    int A(long& a){return 2*a;}
    int A(const int& a){return 3*a;}//const参数重载
    Test& B(int& a) { cout << "type" << endl; return *this; }
    const Test& B(int& a) const { cout << "const type"<<endl; return *this; }//const成员函数重载
    };

const参数重载

const参数保护传递过来的参数，使其不会被改变（所以值传递的参数使用const没有保护的作用，因为值传递的参数的改变不会改变原来参数的值）。
值传递的const参数重载会报错。
有const参数修饰的参数可以传入const值和非const值。
没有const参数修饰的参数不能传入const的值。

void main(){
    const int& num1=1;
    int x=1;
    int& num2=x;
    Test t;
    cout<<t.A(num1)<<endl;
    cout<<t.A(num2)<<flush;
}

输出：
3
1

const成员函数重载

成员函数重载可以看作时对this指针进行const修饰的过程。
根据调用重载函数的对象类型，来决定返回的是const类型的this指针还是没有修饰的指针。

Test t1;
const Test t2;
int y = 100;
int& z = y;
t1.B(z);//非const类调用非const成员函数
t2.B(z);//const类调用const成员函数

输出：
type
const type
